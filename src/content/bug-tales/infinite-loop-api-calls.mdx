A `useEffect` without a dependency array caused uncontrolled refetching on every render.

## Steps to Reproduce

1. Open the user dashboard
2. `useEffect` triggers API call
3. Response updates state
4. State change retriggers `useEffect`
5. Repeat â†’ flood of requests

## Solution

```tsx
// Before
useEffect(() => {
  fetchUserData(user.id)
})

// After
useEffect(() => {
  fetchUserData(user.id)
}, [user.id])
```

Additional fixes included request debouncing, a circuit breaker to pause retries, and `AbortController`-based cancellation.
